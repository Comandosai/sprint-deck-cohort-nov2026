AVPS — где будет жить твой сотрудник

Сервер в облаке, на котором OpenClaw работает 24/7. Не зависит от твоего ноутбука и интернета дома. На воркшопе тебе понадобятся всего две вещи: IP-адрес и root-пароль — придут на email от хостера сразу после оплаты.

Минимальные требования

  • 4 GB RAM (не меньше 2)
  • 20–40 GB SSD
  • Ubuntu 24.04 LTS
  • KVM (НЕ OpenVZ — ломает swap)
  • Публичный IPv4
  • Трафик от 1 TB/мес
  • Datacenter в Европе

CTelegram — бот и твой User ID

Бот — это «лицо» твоего сотрудника в Telegram. Создаёшь нового через @BotFather (5 минут). User ID нужен чтобы бот отвечал только тебе.

Открой @BotFather в Telegram
Найди в поиске → нажми /start. Это официальный бот для создания других ботов.
Создай нового бота
Команда /newbot → введи имя (отображается, можно русское) → введи username (латиницей, должен заканчиваться на _bot, например my_assistant_bot).
Сохрани токен
@BotFather пришлёт строку вида 7891234567:AAH.... Это TELEGRAM_BOT_TOKEN — никому не показывай. Скопируй в Notes.
Включи Privacy mode (рекомендуется)
У @BotFather: /mybots → выбери своего → Bot Settings → Group Privacy → Enable. Чтобы бот не читал чужие сообщения если случайно попадёт в группу.

Открой @userinfobot → нажми /start → бот пришлёт твой числовой ID (например 123456789). Это TELEGRAM_USER_ID — без него бот будет отвечать всем подряд.

antigravity · chat
Промпт 4: Подготовь VPS по разделу A стандарта. Подключайся как root через ~/.ssh/clawd_ed25519. Выполни: 1. apt update && apt upgrade -y 2. ✓ Работает у всех — $VPS_IP берётся из твоего .env
✓ Не зависит от имени папки проекта (главное чтобы ты был В папке с .env)
✓ ~/.ssh/clawd_ed25519 и clawd — стандартные значения из Промптов 2 и 4

⚠️ Просит пароль = ключ не подхватился. Скорее всего опечатка в команде. Проверь что VPS_IP заменён на реальный IP. Не вводи пароль (его нет) — нажми Ctrl+C.

Зачем нужен
Второй чат AI = независимый эксперт

Главный AI в чате установки — исполнитель. Когда ты застрял, его легко уговорить «попробовать что-нибудь» (переключить канал, переустановить). Это путь к 2-дневному аду.

Консультант в отдельном чате только консультирует — читает базу знаний и интернет, даёт точный ответ. Не лезет в SSH, не меняет VPS. Не унаследовал bias главного AI.
