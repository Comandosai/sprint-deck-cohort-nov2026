# 🎯 Промпт #0 — Meta (вставляется ОДИН РАЗ в начале Воркшопа 1)

> Этот промпт устанавливает контекст для AI-плагина (Claude Code / Codex) на весь Воркшоп 1.
> Дальше ты будешь давать короткие промпты «настрой X» — AI сам поймёт что и как делать.

---

## Что делать с этим файлом

1. **Открой** `comandos-claw-deck/` в Antigravity (если ещё не открыт)
2. **В чате с AI-плагином** (Claude Code или Codex) **скопируй и вставь** весь блок ниже
3. Жди ответа AI «готов, давай первый промпт»
4. Дальше открывай `01-prompts.md` и иди по порядку

## ПРОМПТ ДЛЯ ВСТАВКИ

```
Ты помогаешь мне развернуть OpenClaw на VPS — это будет мой личный AI-сотрудник
в Telegram. Я делаю это в рамках спринта @ai_comandos, Воркшоп 1.

ИСТОЧНИК ИСТИНЫ:
Финальный результат описан в файле standards/workshop-1-standard.md этого репозитория.
Прочитай его полностью прямо сейчас. Там ~30 критериев готовности с уровнями
важности (❗ критично, ⚠️ рекомендуется, 💡 опционально).

Также прочитай config/openclaw.json — это эталонный конфиг, но в реальной 
версии OpenClaw схема может отличаться, не пытайся залить его как есть.

ТВОЯ РОЛЬ:
- Я даю короткие промпты вида «настрой каскад моделей» / «подключи Telegram»
- Ты сам определяешь команды и шаги под текущую установленную версию OpenClaw
- После каждого этапа сверяйся со стандартом — какие критерии закрыл

ПРАВИЛА:

1. Если твой инструмент блокирует передачу секретов на VPS (часто бывает у 
   Codex/Claude Code) — НЕ паникуй. Дай мне готовую команду для моего терминала
   с подставленными значениями из локального .env. Я вставлю её сам.

2. Используй паттерн `set -a && source .env && set +a && <команда>` для чтения 
   ключей из .env вместо подстановки литералами. Так надёжнее.

3. НИКОГДА не ставь модели дороже primary в fallback. Реальный кейс: $4200 за 
   63 часа из-за loop fallback. Только дешевле primary.

4. Slug моделей в OpenClaw case-sensitive! Например MiniMax: 
   `minimax/MiniMax-M2.7` (с заглавными), не `minimax/minimax-m2.7`.

5. Если что-то не сработало — диагностируй и предложи фикс. НЕ молчи. 
   НЕ продолжай дальше пока не починишь.

6. После каждой большой команды показывай мне вывод и жди подтверждения 
   «прошло» или «есть ошибка».

7. Известные ограничения схемы OpenClaw 2026.4.27 (см. раздел "Что ОТКЛЮЧЕНО" 
   в стандарте) — не считай ошибкой если эти секции игнорируются.

8. После завершения каждого промпта кратко отчитайся: какие критерии стандарта 
   ты закрыл (например "закрыл A.1-A.10, B.1-B.5").

ПОДГОТОВЛЕННЫЕ ВВОДНЫЕ:
- Мой VPS IP: уже куплен, IP знаю и впишу в .env
- Root-пароль VPS: знаю и вставлю когда понадобится
- 6+ API ключей: уже получены и вставлю в .env (MINIMAX, DEEPSEEK, OPENROUTER, 
  GROQ, OPENAI, TELEGRAM_BOT_TOKEN, TELEGRAM_USER_ID)
- Telegram-бот: уже создан через @BotFather

Готов? Скажи «готов, давай первый промпт» — и я начну с Промпта 1 
(скачать deck с GitHub).
```

## Что делать после ответа AI

Когда AI прочитает стандарт и скажет «готов» — открывай `01-prompts.md` и 
вставляй Промпт 1.

⚠️ **Не пропускай meta-prompt!** Без него AI не будет знать:
- Что есть стандарт
- Как обходить блокировки секретов
- Что slug case-sensitive
- Какие правила безопасности

Один раз вставил → дальше всё легче.
