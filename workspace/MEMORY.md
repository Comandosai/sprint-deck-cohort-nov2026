## Гигиена памяти

1. Перечитать MEMORY.md.
2. Удалить устаревшее.
3. Объединить дубликаты.
4. Перенести «факт временный» → daily log, «факт постоянный» → MEMORY.md.

См. skill `consolidate-memory` (если установлен).
