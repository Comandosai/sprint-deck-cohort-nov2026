## 🎯 Цель блока

После этого блока у тебя:

1. **Smart Routing** в `openclaw.json` — каждый запрос автоматически идёт в нужную модель (Opus для ризонинга, Sonnet для основной работы, Flash-Lite для heartbeat и классификации, DeepSeek для дешёвых subagent'ов). Цель — **64-92% экономии** API без потери качества.
2. **Локальный Whisper API в Docker** — голосовые из Telegram транскрибируются бесплатно (faster-whisper INT8 на CPU 4GB VPS). Резерв — OpenAI Whisper API ($0.006/мин).
3. **Memory Injection** — `USER.md` (60-150 строк ядра) жёстко вшивается в каждый системный промпт. Это разница между «агент знает что я работаю в Telegram» и «агент каждый раз спрашивает».
4. **Cognee** как Knowledge Graph для смысловой памяти на годы (entities + relations + temporal edges).
5. **Memsearch** как cold-archive поверх Markdown + Milvus — индексация тысяч файлов из Obsidian-vault.
6. **Hybrid retrieval + reranker** — Voyage rerank-2 / bge-reranker-v2 поверх vectors+BM25+SPLADE.
7. **Eval-pipeline** на DeepEval/RAGAS — автоматическая проверка регрессий после каждого изменения промптов.

Это финал спринта — после блока 20 у тебя нет «дыр» в архитектуре.

## 💡 Лайфхаки и про-приёмы (15 штук)

1. **Не используй один классификатор на всё.** Разнеси: (a) determination сложности (`heuristic` по длине + наличию слов «архитектура/refactor/plan» → Opus, иначе Sonnet); (b) language detection для Whisper (RU vs EN); (c) intent classification (вопрос/команда/мысль). Каждый — отдельный дешёвый Flash-Lite вызов с кэшем.

2. **Heartbeat на Flash-Lite, а не на Sonnet.** Heartbeat — это служебный пинг агенту каждые 30 минут, чтобы он «сверился» с задачами. Платить Sonnet'у за это — слив. `heartbeat.model = google/gemini-2.5-flash-lite` экономит ~$30-60/мес на активном инстансе.

3. **Subagents — на DeepSeek Reasoner.** Цена $2.74/M tok против Opus $15/$75. Качество ризонинга для подзадач сопоставимо. `subagents.maxConcurrent = 1` важен, чтобы не словить rate-limit.

4. **Memory Injection — короче и плотнее.** USER.md → не больше 80-120 строк. Каждая строка стоит токенов в каждой сессии. Структура: `IDENTITY (3 строки) → ENVIRONMENT (5 строк) → PREFERENCES (10 строк) → ACTIVE PROJECTS (15 строк) → DO/DONT (10 строк)`. Всё остальное — в Cognee/Memsearch (lazy-load).

5. **Prompt caching обязателен.** Если USER.md инжектится — она должна быть в `cache_control: ephemeral` блоке Anthropic. Экономия 30-99% на повторных вызовах. Cache hit в OpenClaw настраивается в `agents.defaults.caching.enabled = true`. [VERIFY]

6. **Cognee `cognify()` — батч, а не stream.** Первая инициализация на 1000 markdown-файлов — это **тысячи embedding-вызовов**. На OpenAI text-embedding-3-small это ~$0.50-2.00 за прогон. Делай ночью, batch_size=64, через `--profile postgres` для надёжности.

7. **Memsearch + Obsidian Vault — чудо-сетап.** Memsearch следит за `~/ObsidianVault/**/*.md`, инкрементально хеширует SHA-256, не пересчитывает unchanged блоки. Подключи `memsearch config set milvus.uri http://localhost:19530` (локальный Milvus в docker-compose) — нулевой рекурринг.

8. **Reranker экономит токены сильнее vector-поиска.** Top-50 от vector + top-10 от reranker даёт лучший recall, чем top-20 от vector. `bge-reranker-v2-m3` локально на CPU — ~50ms на 50 doc'ов, ноль API-стоимости. Cloud-вариант Voyage rerank-2 = $0.05/1k запросов.

9. **Hybrid retrieval = vectors + BM25 + (optionally) SPLADE.** Vectors ловят семантику («как настроить безопасность?» ↔ заметка про fail2ban), BM25 ловит точные имена/коды (`UFW-rule-4242`), SPLADE — гибрид. Слияние через RRF (Reciprocal Rank Fusion, k=60). Memsearch делает это из коробки.

10. **Whisper — чанкуй до 10 сек.** На 4GB VPS большие файлы дают OOM (memory spike до 4.2GB). Чанк 10s + INT8 квантование = sub-3s latency на CPU, стабильно. Используй `vad_filter=True` для пропуска тишины — экономия 20-40% времени.

11. **Whisper distil-large-v3-en только для английского.** Для русского — `large-v3` или `large-v3-turbo` (быстрее, ~50% ресурсов, accuracy near-large-v3). [VERIFY] на твоих голосовых.

12. **Self-improvement через nightly cron.** В 3:00 запускается агент с моделью DeepSeek Reasoner: читает логи дня, вытаскивает 3-5 ошибок/инсайтов, добавляет в `SOUL.md` секцию `## Lessons learned YYYY-MM-DD`. Не больше 5 пунктов в день, чтобы не раздуть.

13. **Eval-pipeline в CI.** DeepEval/RAGAS прогоняет 30-50 «золотых» запросов после каждого изменения USER.md/SOUL.md. Метрики: faithfulness, answer_relevancy, context_precision. Регрессия > 5% — блокирует merge. Стоимость ~$0.10-0.30 за прогон на Haiku-judge.

14. **Moderation-bracket на вход и выход.** Дешёвый Haiku 4.5 ($1/$5 per M) проверяет (a) входящий текст из Telegram на prompt-injection, (b) исходящий ответ — на утечку секретов. Это критично, если бот публичный. Не заменяет Constitutional Classifiers, но дополняет.

15. **Cluster через Mem0 sync, а не через шардинг.** Если поставишь второй VPS — не разноси Qdrant по нодам, лучше держи единый Mem0-сервер на основной ноде, а второй VPS дёргает её через `MEM0_API_URL`. Шардинг embedding'ов на 2 узла без сетевой шторм-защиты — головная боль, выигрыш минимален.

**Бонус-лайфхак (16):** **Latency stack** для голосового ассистента — Cartesia Sonic 3 (TTS, 95ms TTFA) + Whisper-streaming (STT) + prompt caching (TTFT -80%) + Sonnet с `stream=true`. Чувство «реального диалога» вместо «отправил-подождал-получил».


### 6) Reranker (локальный) — bge-reranker-v2-m3

```python
# pip install FlagEmbedding
from FlagEmbedding import FlagReranker
reranker = FlagReranker('BAAI/bge-reranker-v2-m3', use_fp16=False)  # CPU-mode

def rerank(query, docs, top_k=10):
    pairs = [[query, d] for d in docs]
    scores = reranker.compute_score(pairs, normalize=True)
    return sorted(zip(docs, scores), key=lambda x: -x[1])[:top_k]
```

## ⚠️ Подводные камни

1. **Smart Routing → молчаливая деградация качества.** Если классификатор Flash-Lite ошибочно роутит сложный архитектурный вопрос в Sonnet — ответ хуже, но без ошибки. **Лечение:** недельный лог `model_used` и точечный sampling 5% запросов в «обе модели одновременно» с DeepEval-сравнением.

2. **Memory Injection → exploding context.** USER.md разрастается → токены в каждом запросе → счёт растёт. **Лечение:** жёсткий лимит 4000 токенов (5-7% от 200k context window). Всё что выше — в Cognee.

3. **Cognee первая инициализация = шторм API.** 10k файлов = 10k LLM extraction + 10k embedding. На rate-limit OpenAI это часы. **Лечение:** `parallel=4`, `batch_size=32`, ночью, не на проде, retry с exp-backoff.

4. **Memsearch + Milvus = ещё один сервис в Docker.** RAM 512MB-1GB. На 4GB VPS уже впритык (OpenClaw 1GB + Whisper 1.5GB + Mem0/Qdrant 1GB + Milvus 1GB = 4.5GB → swap-thrash). **Лечение:** либо vendor Milvus в Zilliz Cloud (есть free tier), либо отказ от Memsearch и оставить только Cognee.

5. **Whisper + русский = качество хуже EN.** На large-v3 WER для русского ~7-12% против ~3-5% EN. **Лечение:** в промпте Whisper передавай `initial_prompt` с типичной лексикой («OpenClaw, Sonnet, embedding, Cognee»), это снижает WER на 20-30%.

6. **Memory Injection vs prompt-injection-атаки.** Если в USER.md прокрался mailicious-блок (через self-improvement!), он будет прилетать в каждый промпт. **Лечение:** все автоматические записи в SOUL.md/USER.md проходят через Anthropic moderation API + ручной gate по объёму ($> 100 строк за день = алерт).

7. **Reranker latency.** Cohere/Voyage cloud ~150-300ms — нормально, но на streaming-голосе заметно. Локальный bge-reranker-v2-m3 на CPU 4GB ~80-150ms на 50 doc'ов. **Лечение:** реранкуй только если top-1 vector score < 0.75 (порог уверенности).

8. **DeepSeek R1 для subagent → выпуски модели нестабильны.** Известна история кратких outage'ов и «китайская цензура» на политические темы. **Лечение:** fallback в `subagents.fallbacks: ["openai/o3-mini","anthropic/claude-haiku-4-5"]`.

9. **Self-improvement → дрейф личности.** Через 3 месяца SOUL.md накопит 200+ Lessons и агент станет «другим человеком». **Лечение:** ежемесячный ручной review SOUL.md (≤ 30 минут) + git-история (откат при регрессии).

10. **Eval-pipeline стоит денег.** 30 запросов × Haiku-judge × 3 метрики ≈ $0.15 за прогон. 10 прогонов в день = $45/мес. **Лечение:** запускай только на изменении промптов (git pre-commit), не на каждый коммит кода.

**Smart Routing:**
```bash
# Симулируй 20 запросов разной сложности через openclaw exec

jq -r '.model' /var/log/openclaw/access.jsonl | sort | uniq -c
# Ожидание: ~70% sonnet, ~10% opus, ~15% lite, ~5% ds-r1
```

**Whisper:**
```bash
time curl -F "audio_file=@30sec.ogg" "http://127.0.0.1:9000/asr?language=ru"
# Ожидание: <8 сек на 30-сек аудио, large-v3-turbo INT8 CPU
```

**Memory Injection:**
- Спроси у бота: «В каком часовом поясе я работаю?» — без поиска, ответ моментальный (из системного промпта).
- Проверь usage.input_tokens — должен включать +N токенов USER.md.
- Cache hit: при повторном запросе `cache_read_input_tokens` > 90% от cached.

**Cognee:**
```python
# Должен вернуть relations не только text-similarity
await cognee.recall("кто из моих контактов работает с Telegram?", search_type="GRAPH_COMPLETION")
```

**Memsearch:**
```bash
memsearch search "smart routing config" --top-k 5
# Должны вернуться файлы блока 20 с RRF-score > 0.6
```

**Eval:**
```bash
deepeval test run tests/eval_user_md.py
# Ожидание: passed >= 27/30, faithfulness avg > 0.85
```

**End-to-end smoke:**
- Голосовое 15 сек в Telegram «напомни мне что я обсуждал про cognee на прошлой неделе» →
- Whisper (~3 сек) → router → Sonnet → memorySearch + cognee.recall → reranker → ответ → Cartesia TTS → голосовое в Telegram.
- **Целевая P50 latency: 6-9 сек.** P95 < 15 сек.
